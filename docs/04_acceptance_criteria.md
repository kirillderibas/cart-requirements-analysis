# 04 — Acceptance Criteria (критерии приемки) + базовые сценарии

## Лимит на позицию (1…10)
AC-01: Пользователь может добавить новый товар в корзину с количеством 1…10.
AC-02: Если товар уже в корзине, при добавлении N количество увеличивается, но не превышает 10.
AC-03: При попытке увеличить количество позиции выше 10 действие отменяется, показывается сообщение про лимит 10.

## Лимит разных позиций (<=5)
AC-04: Пользователь может иметь в корзине до 5 разных SKU.
AC-05: При попытке добавить 6-ю позицию действие отменяется, показывается сообщение про лимит 5 разных товаров.

## Лимит суммарного количества (<=20)
AC-06: Пользователь может иметь суммарно до 20 единиц товаров в корзине.
AC-07: При попытке превысить 20 действие отменяется, показывается сообщение про лимит 20.

## Изменение количества и удаление
AC-08: Количество позиции можно изменить только в диапазоне 1…10.
AC-09: Удаление товара из корзины осуществляется кнопкой "Удалить"; позиция исчезает из списка.

## Отображение корзины
AC-10: Для каждой позиции отображается: название, количество, цена за единицу, сумма позиции.
AC-11: Отображается итоговая сумма корзины как сумма всех позиций.

## Цены (актуальная цена из каталога)
AC-12: При открытии корзины цены обновляются из каталога и выполняется пересчет суммы.
AC-13: Если цена изменилась, пользователь видит уведомление об изменении и пересчете.

## Реклама/рекомендации
AC-14: На странице корзины отображается блок "Рекомендуем" (если включен).
AC-15: Контент блока обновляется по будням в 09:00 и 18:00 (Europe/Moscow).
